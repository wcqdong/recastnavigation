
Windows
Download SDL Developer Libraries from http://www.libsdl.org and unzip them to SDL/

1. 打开sdl工程
	RecastDemo\Contrib\SDL\VisualC\SDL.sln
2. 编译成目标系统的位数和模式，例如x64 | Release
3. 把SDL2.dll、SDL2.lib、SDL2main.lib拷贝到RecastDemo依赖的目录，例如
	RecastDemo\Contrib\SDL\lib\x64\
	或
	RecastDemo\Contrib\SDL\lib\x86\
	

OSX
Download and install OSX SDL Developer Libraries from http://www.libsdl.org
